# NextJS + TailwindCSS + MongoDB + Cloudinary + Google OAuth2

Un Kit Starter complet NextJS 15, fonctionnel avec MongoDB, Cloudinary, TailwindCSS, Google OAuth2

## Stack Technique
NextJS, MongoDB, Cloudinary, TailwindCSS

## Installation

### Étape 1: Depuis le dossier extrait
```bash
bash setup.sh
```

### Étape 2: Installation des dépendances
```bash
npm install
```
